## âš™ï¸ Installation

To run **CodeBridge** on your local machine:

1. **Clone this repository:**

    ```bash
    git clone https://github.com/SharkyBytes/CodeBridge.git
    ```

2. **Set up the Backend:**

    - Navigate to the `server` directory:

    ```bash
    cd server
    ```

    - Install dependencies:

    ```bash
    npm install
    ```

    - Run the server:

    ```bash
    npm run dev
    ```

    The backend will be available on `http://localhost:5000`.

3. **Set up the Frontend:**

    - Navigate to the `client` directory:

    ```bash
    cd client
    ```

    - Install dependencies:

    ```bash
    npm install
    ```

    - Run the frontend:

    ```bash
    npm run dev
    ```

    The frontend will be available on `http://localhost:5173`.

4. **Access the Application:**
   Open your browser and navigate to `http://localhost:5173` to start coding collaboratively.
